About

This is an example of the Flickr API. The API is called via Javascript with the JQuery library Ajax function. The flickr.photos.search method is used but can be swapped for any other. A Bootstrap template was used to display the photos returned from the call; although it needed to be altered so that it is responsive for different size screens.
